Upload a research PDF, highlight passages, add notes, ask follow-up questions, and export an annotated PDF — no library or account required.

## What it does

- Upload any PDF and parse it for selectable text (single- and two-column layouts)
- Highlight text or drag a region over figures and equations
- AI-assisted annotations and threaded follow-up chat on each note
- Export a PDF with your highlights and sticky-note comments burned in

## Stack

FastAPI · PyMuPDF · React · OpenAI (optional) · Railway API + Netlify UI
