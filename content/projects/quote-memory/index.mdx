Half-remember a line from a show? **Quote Memory** finds the episode and timestamp from fuzzy subtitle search across multiple series.

## What it does

- Ingests subtitle tracks and builds a searchable index
- Handles partial, misremembered, or approximate quotes
- Returns ranked matches with episode + timestamp

## Stack

FastAPI · fuzzy matching · subtitle indexing · static deploy (Railway/Netlify)
